### FE

1. git clone
2. yarn
3. yarn start

### DB

1. docker mysql

```
docker run -p 3306:3306 --name todolist -e MYSQL_ROOT_PASSWORD=1234 -e MYSQL_DATABASE=todolist -e MYSQL_USER=dyne -e MYSQL_PASSWORD=1234 -d mysql
```
